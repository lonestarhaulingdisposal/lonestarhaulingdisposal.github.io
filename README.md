# Lone Star Hauling & Disposal Premium Site

This package contains a polished GitHub Pages-ready website for Lone Star Hauling & Disposal.

## Files
- index.html
- styles.css
- script.js
- logo-source.png

## Before publishing
Replace:
- (000) 000-0000
- info@example.com

## Publish on GitHub Pages
1. Create or open your public GitHub repository.
2. Upload all files in this folder.
3. Go to Settings > Pages.
4. Set:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save and wait about a minute.
